Algorithmes géométriques
========================

Le fichier cgeom_s21.jar contient une application permettant de vérifier
visuellement le fonctionnement des méthodes de la classe Geom.

Sous Eclipse, importer cgeom_s21.jar et l'ajouter dans les ressources
du projet (Project->Properties->BuildPath->Add Jar). Puis exécuter le 
programme de la classe s21.CG.

La librairie cgeom_s21.jar suppose que vos sources (Geom.java, 
PointComparator.java et Segm.java) sont dans le package s21.
